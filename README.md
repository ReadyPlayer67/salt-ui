## 特点

- ⚡ 基于pnpm和vite构建
- 🔥 vue3 + TSX语法
- 🎨TailwindCSS + sass的样式体系
- ⚙️使用Vitest进行单元测试
- 🔨eslint + prettier + husky，完整的代码校验体系
- 📜基于vitepress的自动化文档
- 🔳支持使用命令行创建组件

### 创建组件
```bash
cd cli
pnpm run cli -t component
```

### 打包组件库
```bash
pnpm run build:component
```
